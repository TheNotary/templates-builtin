## Installation

This is a simple HTML project that includes a CSS and Javascript file.  Out of the box, you don't need to compile or do anything, just `open index.html` to get hacking.  Alternatively consider serving the project via (some APIs require this --archaic, I know):

```
npx serve
```

Otherise, if you want to use the VS Code debugger, press the play button in VS Code to make use of the included `.vscode/launch.json` and debug in an IDE.

## Adding Dependencies

You don't need to touch NPM for anything to work, but `npm init` has already been run for this project, so you can do `npm install some-dependency` and source it into your HTML or javascripts.
